### `npm install`

Installs all the dependencies that were used to build this project.


### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.



### NOTE:

    Please make sure flask app is running of port 5000 as thats what I used to run on my local. 
    If flask app is running on any other port please update the port json file in Config folder configurations.
